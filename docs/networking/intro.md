# Networking

## What is Networking

- Networking is the process of connecting devices to share data  
- allows communication between systems  
- used in internet, local networks, and cloud  

---

## Types of Network

- LAN → Local Area Network  
- WAN → Wide Area Network  
- MAN → Metropolitan Area Network  

---

## Basic Components

- router  
- switch  
- firewall  
- server  
- client  

---

## Data Transmission

- data is sent in the form of packets  
- each packet travels through network  

---

## Use Case

- internet communication  
- file sharing  
- remote access
