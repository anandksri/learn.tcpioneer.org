# Networking Commands

## Check IP

```
ip a
```

---

## Ping

```
ping google.com
```

---

## Traceroute

```
traceroute google.com
```

---

## DNS Lookup

```
nslookup google.com
```

---

## Netstat

```
netstat -tuln
```

---

## Curl

```
curl http://example.com
```

---

## Use Case

- troubleshoot network
- check connectivity
- analyze traffic
