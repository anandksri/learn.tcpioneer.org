# IP Addressing

## What is IP Address

- unique identifier for a device on network  
- used to identify source and destination  

---

## Types of IP

- IPv4 → 192.168.1.1  
- IPv6 → longer format  

---

## Private IP Range

- 192.168.x.x  
- 10.x.x.x  
- 172.16.x.x  

---

## Public IP

- accessible over internet  

---

## Check IP

```
ip a
```

---

## Use Case

- identify devices  
- routing data
