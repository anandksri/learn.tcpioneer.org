# Basic Docker Commands

## Pull Image

```
docker pull <image-name>
```

---

## Run Container

```
docker run <image-name>
```

---

## List Containers

```
docker ps
```

```
docker ps -a
```

---

## Stop Container

```
docker stop <container-id>
```

---

## Remove Container

```
docker rm <container-id>
```

---

## Remove Image

```
docker rmi <image-id>
```

---

## Execute Command in Container

```
docker exec -it <container-id> bash
```
