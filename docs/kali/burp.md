# Burp Suite

## What is Burp Suite

- Burp Suite is a web application security testing tool  
- used to intercept and modify HTTP/HTTPS requests  
- widely used in web security and CTF  

---

## Start Burp Suite

```
burpsuite
```

---

## Setup Proxy

- set browser proxy to 127.0.0.1:8080  
- open browser through Burp  

---

## Intercept Request

- turn intercept on  
- capture request from browser  

---

## Send to Repeater

- right click request  
- send to repeater  

---

## Send to Intruder

- right click request  
- send to intruder  

---

## Use Case

- test web applications  
- modify requests  
- find vulnerabilities
