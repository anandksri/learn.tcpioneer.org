# Basic Linux Commands

## Navigation

```
cd → change directory
cd .. → go back
pwd → present working directory
```

---

## List Files

```
ls → list files
ls -l → detailed list
ls -a → hidden files
```

---

## File Operations

```
touch file.txt → create file
mkdir folder → create directory
rm file.txt → remove file
rm -r folder → remove directory
```

---

## Copy and Move

```
cp file1 file2 → copy
mv file1 file2 → move/rename
```

---

## View Files

```
cat file.txt → view content
less file.txt → scroll view
head file.txt → first lines
tail file.txt → last lines
```

---

## Search

```
grep "text" file.txt
```

---

## Clear Terminal

```
clear
```
