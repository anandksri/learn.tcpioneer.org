# File System and Permissions

## Linux File System

- / → root directory  
- /home → user files  
- /etc → configuration files  
- /var → logs and data  
- /bin → system binaries  

---

## File Permissions

r → read  
w → write  
x → execute  

---

## Permission Format

-rwxr-xr--  

- first part → file type  
- next 3 → owner  
- next 3 → group  
- last 3 → others  

---

## Change Permissions

chmod 755 file  

---

## Change Ownership

chown user file
