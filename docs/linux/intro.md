# Linux

## What is Linux

- Linux is an open-source operating system  
- used to manage hardware and software  
- widely used in servers, cybersecurity, and development  

---

## Why Linux

- free and open-source  
- secure  
- stable  
- highly customizable  
- widely used in hacking and cybersecurity  

---

## Linux Distributions

- Ubuntu  
- Kali Linux  
- Debian  
- Fedora  

---

## Where Linux is Used

- servers  
- cloud systems  
- cybersecurity  
- embedded systems  

---

## Kernel

- core part of the operating system  
- manages hardware and system resources
